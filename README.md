# 🦄 MLP TCG Collection Vault

A fan-made collection manager for the **Kayou My Little Pony Trading Card Game**.

Track your collection, duplicates, wishlists, set completion, release years, and export checklists from a simple and easy-to-use interface.

Designed by **Bow** for collectors who wanted something easier than managing thousands of cards through spreadsheets.

# ⚠️ Backup Recommendation

Most of MLP TCG Collection Vault runs locally.

Your collection data is stored on your device.

It is strongly recommended that you regularly export backups.

Backups protect your collection if:

* Browser data is cleared
* Files are updated
* Local storage is reset
* You move to a new computer
* Future versions change the storage format
