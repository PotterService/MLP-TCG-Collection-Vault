About / Donate

MLP TCG Collection Vault is a fan-made local checklist and collection interface designed by Bow for collectors who want an easier way to track Kayou My Little Pony cards.

About This Project

This project was designed by Bow. The goal is to turn a large collector checklist into a simple GUI interface where collectors can search sets, mark cards owned, track duplicates, export checklists, and keep a local backup of their collection.

I hope to keep updating this project over time as new information, better photos, and new checklist updates become available.

The online version of this project can be found at: MLP.PotterService.com.

A standalone Windows desktop application is currently in development.

Current Version

Current Release: V11

Recent updates include year filtering, set information pages, collection statistics, exportable checklists, backup tools, and the About / Donate page.

Backups Are Important

Please export and backup your collection regularly. Most of this site runs locally in your browser, which means future updates, browser changes, clearing site data, or replacing files could affect your local database. Backups make it much easier to import your collection again if something breaks.

Photos and Images

I am currently working on future photo updates. At the moment, you can add image URLs to individual cards. This helps save local storage space because the app does not need to store every card image directly on your computer.

Around 90% of this site runs locally. Your owned cards, duplicate counts, notes, and image URLs are saved in your browser unless you export a backup.

Community Checklist Note

Goda’s post explained that finding all the cards, collecting the information, and making a clean checklist was very time consuming. They also noted that there may be a few mistakes because of how difficult the project was.

The checklist includes many sets such as Moon versions, Rainbow versions, Fun Moments, Adventure Trip, Pony Life boxes, Equestria Girls, extra cards, starter decks, Fantasy Wonderland, Discord Moment, Nightmare Night, Dispel Cold, MLPG promotional cards, and other extra cards.

Some card names may still use original Chinese names where a correct name was not available. In those cases, using the card ID online can help identify the card.
