# MLP Kayou Collection Tracker V10

Updates added:

- Year filter
- Home page with set names and release dates, like Rainbow Box 1 / Release Date: June 2021
- Export checklist for selected set
- Info page crediting Bow and Goda
- Reminder to export regularly so local data can be restored after updates
- Keeps the Lorcana-style grid collection view

## Important

The tracker saves collection changes locally in your browser. Use **Backup JSON** and **Export Collection JSON** often.

The card database was generated from the uploaded MLP All Kayou Sets Collector Checklist spreadsheet.
